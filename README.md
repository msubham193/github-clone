
# Github Clone

A Github repository listing page using react , tailwind css and Github Api





## Run Locally

Clone the project

```bash
  git clone https://github.com/msubham193/github-clone.git
```

Go to the project directory

```bash
  cd github-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
